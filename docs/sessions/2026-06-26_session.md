beam = SimpleBeam(span_ft, E_ksi, I_in4, n_elements=20)

Sign conventions:
- V: positive = left face up / right face down

### 7. src/calcs/aisc360.py — COMMITTED (commit 8479114)
AISC 360-22 flexure (Sec. F2/F3) and shear (Sec. G2) for W-shapes.

```python
result = check_beam(
    section_label,   # e.g. "W16X40"
    Mu_kip_ft,       # factored flexural demand (LRFD)
    Vu_kips,         # factored shear demand (LRFD)
    Lb_ft=0.0,       # unbraced length (0 = fully braced)
    Fy_ksi=50.0,     # default A992
    Cb=1.0,          # moment gradient factor (1.0 = conservative)
)
# result.flexure: phiMn, DCR, limit_state, Lp_ft, Lr_ft, code_ref
# result.shear:   phiVn, DCR, Cv1, limit_state, code_ref
# result.summary() — printable check block
```

Limit states covered:
- Sec. F2.1: Yielding (Lb ≤ Lp)
- Sec. F2.2: Inelastic LTB (Lp < Lb ≤ Lr)
- Sec. F2.3: Elastic LTB (Lb > Lr), using Eq. F2-4
- Sec. F3.2(a/b): Flange local buckling (non-compact/slender flange)
- Sec. G2.1(a): Compact web shear (φv=1.0, Cv1=1.0)
- Sec. G2.1(b): Non-compact/slender web shear (φv=0.90)
- NotImplementedError raised for non-compact/slender web in flexure (Sec. F4/F5)

**Verified against AISC 360-22 manual (W16X40, Fy=50 ksi, w=2 k/ft, L=30 ft):**

|---|---|---|---|---|
| 1 | 0 ft | yielding | 273.8 kip-ft | 0.822 |
| 2 | 10 ft | inelastic LTB | 229.0 kip-ft | 0.983 |
| 3 | 20 ft | elastic LTB | 118.7 kip-ft | 1.895 |

Key intermediates confirmed by user:
- Lp = 5.55 ft, Lr = 15.89 ft (matches AISC Table 3-10)
- phiMp = 273.8 kip-ft, phiMr = 170 kip-ft (matches Table 3-10 boundary values)
